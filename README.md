# 🧠 Employee Salary Prediction using Machine Learning

This project predicts employee salaries based on various features such as job role, experience, department, and other HR-related attributes. Built using Python and machine learning algorithms, it demonstrates how predictive analytics can enhance HR decision-making.

---

## 📁 Project Structure

---

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning and Preprocessing
- 🔤 Label Encoding of Categorical Features
- 🤖 ML Model Training (Random Forest, Linear Regression, etc.)
- 📈 Model Evaluation using R², MAE, RMSE
- 🧠 Predict salary class or salary amount
- 🗂️ Save and Load model for real-time usage

---

## 📊 Dataset Description

The dataset includes the following columns:

| Column Name       | Description                           |
|-------------------|----------------------------------------|
| EMPLOYEE_ID       | Unique employee identifier             |
| FIRST_NAME        | Employee's first name                  |
| LAST_NAME         | Employee's last name                   |
| EMAIL             | Company email                          |
| PHONE_NUMBER      | Contact number                         |
| HIRE_DATE         | Date of joining                        |
| JOB_ID            | Job role/position                      |
| SALARY            | Employee salary (Target variable)      |
| COMMISSION_PCT    | Commission percentage (if any)         |
| MANAGER_ID        | Reporting manager ID                   |
| DEPARTMENT_ID     | Department identifier                  |

---

## 🔍 Algorithms Used

We evaluated multiple models:

- ✅ **Linear Regression**
- ✅ **Decision Tree Regressor**
- ✅ **Random Forest Regressor**
- ✅ (Optional) XGBoost, SVR

The best model was selected based on accuracy and error metrics.

---

## 🧪 Model Evaluation Metrics

- R² Score (Coefficient of Determination)
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
